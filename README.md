# Platformer Lab 3 (JavaScript, Canvas)

## Описание
Мини-игра платформер на Canvas API: движение, прыжки, гравитация, коллизии, монеты (очки), 2 уровня, интерфейс управления.

## Управление
- Влево: A или ←  
- Вправо: D или →  
- Прыжок: Space / W / ↑

## Функционал
- Игровой цикл: requestAnimationFrame
- Физика: гравитация, скорость, прыжок
- Коллизии: прямоугольные (AABB) с платформами
- Очки: сбор монет (+10)
- Условия прохождения: собрать все монеты и войти в портал
- 2 уровня
- Best score сохраняется в localStorage
- Кнопки: Старт / Пауза / Рестарт

## Использование нейросети
- Нейросеть: ChatGPT
- Время: ~45 минут (замени на своё реальное)
- Сгенерировано с помощью нейросети:
  - структура проекта
  - классы Game/Player/Platform/Coin/InputHandler
  - логика коллизий и условий победы
  - UI и базовые стили
